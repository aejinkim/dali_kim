# public/assets

Case study media files, organized by project.

## Structure
```
assets/
├── virtuswap/     — VirtuSwap DeFi case study images/videos
├── initia/        — Initia Ecosystem case study images/videos
└── shared/        — Profile photo, OG image, favicon, shared icons
```

## Naming convention
`[project]_[sequence].[ext]`
e.g. `virtuswap_01.jpg`, `initia_03.png`

## Note
Existing files at `/public/` root are legacy — new assets go into subfolders here.
